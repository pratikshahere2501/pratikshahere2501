## 🏆 Featured Projects

### 🏦 RiskVault — Loan Default Prediction Pipeline

End-to-end **Big Data Lakehouse pipeline** processing 255K+ loan records.

- ⚙️ Built on **Apache Spark, Databricks, HBase & Delta Lake**
- 🔄 Automated **ETL with Airflow DAGs** across Bronze/Silver/Gold layers
- 📊 ML-based **loan default prediction** and borrower risk scoring
- 📈 Interactive **Power BI dashboards** for KPI and trend monitoring

**Stack:** `PySpark` `Databricks` `Airflow` `Delta Lake` `Power BI`

### 🧩 Autism Spectrum Disorder Detection System

ML-based system for **early autism screening** using behavioral data.

- 🧪 Compared **Random Forest, SVM & XGBoost** models
- 🎯 Achieved **96%+ prediction accuracy**
- 📐 Evaluated using Precision, Recall, F1-Score, and ROC-AUC
- 🌐 Deployed as a **Flask + SQLite** web app with real-time results

**Stack:** `Python` `Scikit-learn` `Flask` `SQLite`
